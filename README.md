# Predictive Analysis Using Machine Learning (Regression & Classification)

##  Project Overview
This project demonstrates **Predictive Analysis** using **Machine Learning techniques** on a given dataset.  
It applies two supervised learning models:
- **Linear Regression** for predicting continuous numerical values
- **Logistic Regression** for classifying data into categories

The project shows how raw data can be processed, trained, tested, and evaluated to make future predictions.

---

##  Objectives
- To understand predictive analytics using machine learning
- To implement regression and classification models
- To evaluate model performance using appropriate metrics
- To gain hands-on experience with real ML workflow

---

## Machine Learning Models Used
### 1️ Linear Regression
- Predicts continuous output values
- Example: sales prediction, price prediction
- Evaluation metric: **Mean Squared Error (MSE)**

### 2️Logistic Regression
- Predicts categorical output (binary classification)
- Example: High vs Low values
- Evaluation metric: **Accuracy Score**
